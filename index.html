Homework: Array Methods

Task 1: Maasivin metodlarından istifadə edərək [23, 45, 12, 56, 67, 34, 89, 14] bu massivi console-da [89, 67, 56, 45, 34, 23, 14, 12] bu formada yazdır.

let nums1 = [23, 45, 12, 56, 67, 34, 89, 14];
console.log(nums1.sort((a, b) => b-a)); //Output: [89, 67, 56, 45, 34, 23, 14, 12]

Task 2: ["cat", "lion", "dog", "elephant"] massivində uzunluğu 3-dən böyük olan ilk elementi console-da yazdır.

let animals = ["cat", "lion", "dog", "elephant"];
console.log(animals.find((x) => x.length > 3)); //Output: lion

Task 3: [5, 29, -3, 42, -1] massivindəki ilk mənfi elementi console-a yazdır.

let nums2 = [5, 29, -3, 42, -1];
console.log(nums2.find((x) => x < 0)); //Output: -3

Task 4: [21, 24, 3, 67, 85, 60] massivindəki son cüt ədədi console-a yazdır.

let nums3 = [21, 24, 3, 67, 85, 60];
console.log(nums3[nums3.findLastIndex((x) => x % 2 == 0)]); //Output: 60
